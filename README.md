# Bachelor’s Senior Design Project — Horus
Horus is an intelligent emotion and deception analysis system developed as a Bachelor’s Senior Design Project. 
The application uses computer vision and machine learning techniques to analyse human facial expressions and speech 
in order to detect emotions and potential deception.

Horus supports both real-time analysis and offline processing of images and videos. 
The system is available as both a desktop and a mobile application, ensuring cross‑platform accessibility.
